no_auto_helmet (FiveM) — README

Stops GTA’s auto bike helmet. No more auto-equip when getting on motorcycles/bicycles.

Features

✅ Blocks auto-helmet behavior

✅ Instantly removes any auto-spawned helmet

✅ Standalone — works with Qbox/QBCore/ESX

Install (Drag & Drop)

	1.Put the no_auto_helmet folder in resources/.

	2.Add to server.cfg:

ensure no_auto_helmet

	3.Restart the server.

(Optional)

Motorcycles only: in client.lua, change the class check to class == 8.

No dependencies.

FXServer cerulean or newer recommended.
